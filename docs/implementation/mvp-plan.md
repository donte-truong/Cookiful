# MVP Implementation Plan

- monorepo setup
- shared TypeScript config
- Docker placeholders
- app and package boundaries
- environment template

## Phase 1: Core Consumer MVP

- auth and user profiles
- recipe read model
- feed cards
- likes, saves, collections
- search
- cooking sessions
- timers

- meal planning calendar
- recipe scheduling by meal slot
- grocery list generation
- pantry basics
- grocery checklist UX

## Phase 3: Personalization

- better ranking
- notifications
- semantic search
- pantry-aware recommendations

3. Feed, recipe detail, likes and saves
4. Search and collections
5. Cooking sessions and timers
6. AI orchestration v1
